1. Нажмите на раздел:Порты возле терминала

2. Возле https ссылки нажмите на глобус, откроется swagger 

3. Далее нужно авторизоваться -  username: admin ; password: password

4. Затем необходимо создать стол - ввести его номер и вместимость(максимально 20)

5. Чтобы сделать заказ сначала создайте пользователя(/users), появится его id(порядковый номер)

6. В заказе(/bookings) укажите номер стола(table_id), номер пользователя(user_id), дату(год,месяц,число,часы), количество гостей

7. Имеется возможность посмотреть свободные столы(/tables/free) и заказы, отменить заказ(DELETE по номеру заказа) и поменять вместимость стола по его номеру(PUT)
